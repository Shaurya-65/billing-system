# 🧾 Customer Billing System

A simple and interactive billing system built using HTML, CSS, and JavaScript.
It allows users to add products, calculate total cost, and manage items easily.

---

## 🚀 Features

* Add products with price and quantity
* Automatic total calculation
* Delete individual items
* Clear entire bill
* Clean and responsive UI

---

## 🛠️ Technologies Used

* HTML (Structure)
* CSS (Styling)
* JavaScript (Logic)

---

## ▶️ How to Run

1. Download or clone this repository
2. Open `index.html` in any browser
